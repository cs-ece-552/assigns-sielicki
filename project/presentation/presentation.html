CS/ECE552 Spring 2019 Project Report

1 Design Overview

We implemented a small processor implementing the WISC-SP19 ISA, which is a small ISA extension on MIPS.

It is implemented in a 5-stage pipelined design with full forwarding, where data memory and instruction memory are implemented as two instances of a memory system including a 2-way set-associative 4KiB cache in front of 128KiB of total memory.
